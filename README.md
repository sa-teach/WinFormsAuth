# WinForms Authentication App

Windows Forms приложение на C# / .NET с авторизацией, ролями, пазл-капчей и панелью администратора.

## Возможности

- Авторизация с ролями **Администратор** и **Пользователь**
- Интерактивная **пазл-капча** из 4 изображений
- Блокировка аккаунта после **3 неудачных попыток** входа
- Панель администратора: добавление, редактирование, разблокировка пользователей
- База данных **SQLite** (хранится рядом с `.exe`, отдельной установки не требует)
- Пароли хранятся в виде **SHA-256 хеша**

## Быстрый старт

```bash
git clone <repo-url>
cd WinFormsAuth
dotnet run
```

При первом запуске автоматически создаётся база данных с тестовыми пользователями:

| Логин | Пароль | Роль |
|-------|--------|------|
| admin | admin123 | Администратор |
| user1 | user123 | Пользователь |
